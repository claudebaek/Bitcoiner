개인정보 처리방침

최종 수정일: 2026년 1월 19일

비트코이너(이하 "앱")는 사용자의 개인정보를 소중히 여깁니다. 본 개인정보 처리방침은 앱이 수집하는 정보와 그 사용 방법에 대해 설명합니다.

1. 수집하는 정보

1.1 분석 데이터

앱 경험 개선을 위해 Firebase Analytics를 통해 익명의 사용 데이터를 수집합니다:

  • 화면 조회 및 탐색 패턴
  • 앱 기능 사용 통계
  • 기기 유형 및 운영 체제 버전
  • 앱 충돌 보고서

이 데이터는 익명으로 수집되며 개별 사용자를 식별하는 데 사용될 수 없습니다.

1.2 로컬 데이터

다음 설정은 사용자의 기기에만 로컬로 저장됩니다:

  • 언어 설정
  • 자동 새로고침 설정
  • 햅틱 피드백 설정
  • 채굴 계산기 설정

이 데이터는 기기를 떠나지 않으며 어떤 서버로도 전송되지 않습니다.

중요: 비트코이너는 이름, 이메일 주소, 전화번호 또는 금융 정보와 같은 개인 정보를 수집하지 않습니다. 당사는 귀하의 비트코인 지갑 정보에 접근할 수 없습니다.

2. 정보 사용 방법

수집된 익명의 분석 데이터는 다음 목적으로 사용됩니다:

  • 앱 성능 및 안정성 개선
  • 가장 유용한 기능 파악
  • 버그 및 기술적 문제 해결
  • 향후 기능 개발 계획

3. 제3자 서비스

앱은 다음의 제3자 서비스를 사용합니다:

  • Firebase Analytics (Google) - 익명 사용 분석용
  • CoinGecko API - 비트코인 가격 데이터 (개인 정보 공유 없음)
  • Mempool.space API - 블록체인 데이터 (개인 정보 공유 없음)

4. 데이터 보안

수집된 데이터를 보호하기 위해 적절한 조치를 취합니다:

  • 모든 네트워크 통신에 HTTPS 암호화 사용
  • iOS 보안 저장 메커니즘을 사용하여 로컬 데이터 저장
  • 서버에 개인 데이터를 저장하지 않음

5. 사용자 권리

사용자는 다음을 수행할 수 있습니다:

  • 앱을 삭제하여 모든 로컬 저장 데이터 제거
  • iOS 설정에서 "앱이 추적 요청을 허용"을 비활성화하여 분석 거부
  • 데이터 수집에 대한 정보 요청

6. 아동 개인정보

앱은 13세 미만의 아동을 대상으로 하지 않습니다. 당사는 아동의 개인 정보를 의도적으로 수집하지 않습니다.

7. 정책 변경

본 개인정보 처리방침은 수시로 업데이트될 수 있습니다. 변경 사항이 있을 경우 이 페이지에 새로운 개인정보 처리방침을 게시하고 "최종 수정일"을 업데이트하여 알려드립니다.
